# 📚 Course Modules@


## 🟢 1. Foundations (Short but Strong)

- Basic React (components, hooks, API calls)
- Python basics (FastAPI or Flask)
- SQL fundamentals (queries, joins)
- REST APIs

✅ Goal: Students can build a full-stack app quickly.

## 🟣 2. AI Fundamentals for Developers (Important)

Don’t go too deep into math — focus on usage:

- What is AI / ML / LLM
- Prompt engineering basics
- Tokens, context windows
- Model types (GPT, Claude, open-source)
- Embeddings & semantic search

✅ Goal: Developers understand **how to use AI like an API**

## 🔵 3. Working with AI APIs

Hands-on integration:

- OpenAI / Azure OpenAI / other APIs
- Sending prompts from backend (Python)
- Handling responses
- Streaming responses to React UI

Mini project ideas:

- AI text generator
- AI email assistant
- AI chat UI (ChatGPT clone).
